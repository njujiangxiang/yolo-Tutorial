## 🚀 快速开始

### 1. 环境要求

```bash
Python 3.10+
PyTorch 2.0+
GPU (推荐，用于训练)
```

### 2. 安装依赖

```bash
cd yolo-Tutorial
pip install -r requirements.txt
```

### 3. 开始学习

```bash
# 从基础开始
cd 00-setup
python examples/verify_install.py

# 逐步学习每个主题
cd ../01-cv-fundamentals
python examples/basic/image_basics.py
```
